##  Problem Statement
    To calculate a final percentage score for a studend over multiple years and then visualize the results using a user-selected chart type from the matplotlib library

## Scope of the project
    the scope of this program is simply to take a score of student of all four years and represent it in data visualization form
    it also includes the list where total percentage of all four years are displayed

## Target user
    1)students- they can check there end score in form of graphs
    2)teachers- they the see the progress report of the student by just giving the marks input
    3)proggramming beginners- they can understand the basic concept of data visualization and functions of list

## High-level features
    1)weighted marks calculation- mid term(30%),end term(30%),internal(40%)
    2)customizable visualization selection- choose how the final data is presented
    3)simple and easy code for beginners
